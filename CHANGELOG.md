# Changelog

All notable changes to SentinelAI will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-07-15

### Added
- Initial release of SentinelAI
- Monorepo architecture with Turborepo
- Plugin-based security scanner framework
- AI-powered vulnerability analysis and explanation
- Smart contract audit workflow
- Contract monitoring service
- Report generation (Markdown, JSON)
- Web dashboard with dark theme
- Wallet-based authentication (Freighter, Stellar wallets)
- Audit Registry smart contract
- Docker development environment
- CI/CD pipeline with GitHub Actions
- Sample plugins (reentrancy, access control)
- Comprehensive documentation
- API reference

### Security
- JWT-based authentication
- Rate limiting on API endpoints
- Input validation and sanitization
- CORS configuration
